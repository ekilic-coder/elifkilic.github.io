El Salvador

Heat Vulnerability Dashboard & Data Analysis

About This Dashboard

This dashboard shows live and historical heat stress data for El Salvador. Temperature and apparent temperature data comes from ERA5 reanalysis (1960–present) via the Open-Meteo API. Climate projections use CMIP6 HighResMIP models. Heat index is computed client-side using the NWS/Steadman equation. All data loads directly in your browser — no server required.
